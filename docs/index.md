# Count App

> A powerful, extensible counter tracking application for Android, Windows, and Linux

Count App helps you track counts and numeric series quickly and reliably across multiple platforms. Whether you're counting daily habits or tracking measurements over time, Count App provides the tools you need with a beautiful, intuitive interface.

---

## Features at a Glance

| Feature | Description |
|---------|-------------|
| **Multiple Counter Types** | Tap counters for quick counting and series counters for trend tracking |
| **Rich Statistics** | Interactive charts, averages, highs/lows, and time-range filtering |
| **Beautiful UI** | Modern design with dark and light themes |
| **Data Control** | Full JSON export/import for backups and data portability |
| **Extensible Architecture** | Developer-friendly API for adding custom counter types |
| **Cross-Platform** | Native support for Android, Windows, and Linux |

## Counter Types

Count App offers two powerful counter types:

### Tap Counter
**Best for:** Quick counting of discrete events

- Single-tap updates with configurable step size
- Increment or decrement mode
- Optional confirmation dialogs
- **Examples:** Daily water intake, exercise reps, books read

### Series Counter
**Best for:** Tracking values that change over time

- Record numeric measurements with timestamps
- View trends with interactive charts
- Calculate averages, highs, and lows
- **Examples:** Body weight, daily revenue, temperature logs

## Key Capabilities

### Data Management
- **Export** your counters to JSON for backups
- **Import** from previous exports to restore or migrate data
- Full control over your data with human-readable JSON format

### Statistics & Analytics
- Daily, weekly, and monthly trends
- Interactive time-range filtering (1W, 1M, 3M, 1Y, All)
- Update frequency analysis
- Visual charts and graphs

### Customization
- Dark and light theme support
- Configurable counter behavior
- Extensible architecture for developers
