# Quick Start Guide

Get up and running with Count App in minutes!

## Create Your First Counter

When you open the app for the first time, your list will be empty.

1. **Tap the Plus (+)** button in the bottom-right corner.
2. **Choose "Tap Counter"** (simplest option).
3. **Give it a name** (e.g., "Water Glasses").
4. **Tap Save**.

ðŸŽ‰ You've created your first counter!

## Track Something

Now that you have a counter:

1. **Tap the counter card** to increment the count.
2. The number goes up by 1 (or your custom step size).
3. That's it! The app automatically records the time of each tap.

## View Statistics

After you've tapped a few times:

1. **Long-press** the counter card.
2. Select **"View Statistics"**.
3. See your daily activity, trends, and history.
