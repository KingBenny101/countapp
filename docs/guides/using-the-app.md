# Using Count App

A short user guide for everyday use.

## Add a counter

- Tap the floating action button (+) on the home screen.
- Choose a counter type and fill the configuration (name, step size, etc.).
- Save to add it to your list.

## Update values

- Tap a counter to update it (behaviour depends on type):
  - **Tap Counter**: single tap increments/decrements by the step size.
  - **Series Counter**: opens a dialog to enter a numeric value.

## Delete counters

- Long-press a counter to enable selection mode and remove counters.

## View statistics

- Open a counter's info/statistics to see averages, highs/lows, and charts for recent activity.

## Backup & Restore

- Use the app menu to Export Counters to a JSON file and save it externally.
- Import from a previously exported JSON file to restore or move counters to another device.

## Tips

- Regularly export your counters before upgrading major versions.
- Use descriptive names for counters to make statistics readable.
